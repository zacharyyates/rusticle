rusticle

A ranging bow reticle for rust.

How to use it

  1. Start rust in windowed mode.
  2. Start rusticle.

Commands

| Key | Action | -------- | ------ | Pause | Enter/exit adjustment mode | Up | Adjust reticle up | Down | Adjust reticle down | Left | Adjust reticle left | Right | Adjust reticle right | Home | Reset reticle adjustment | End | Exit rusticle
